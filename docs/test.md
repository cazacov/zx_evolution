## 4.2. Операторы каналов

Каждый оператор имеет свою амплитудную огибающую, а также свою частоту, зависящую от основной частоты канала. Частота оператора с точностью до так называемой "расстройки" кратна основной частоте, либо равна 1/2 основной частоты[^3].

Фаза оператора модулируется сигналом на его входе. Сигнал на выходе конкретного оператора описывается формулой:

$$F = A \sin(\omega_{C}t + I \sin \omega_{M}t)$$

[^3]: Канал 3 может использовать полностью независимые частоты для каждого оператора, даже если они не кратны основной (не являются гармониками). Применимо для синтеза ударных инструментов и различных аккордов (однако мажорное и минорное трезвучия можно воспроизвести и на других каналах).



где:
- A - амплитуда (уровень сигнала оператора);
- I - коэффициент модуляции (амплитуда сигнала на входе оператора);
- $\omega_{C}$ - угловая частота (несущая) самого оператора;
- $\omega_{M}$ - угловая частота (модулирующая) на входе оператора.
